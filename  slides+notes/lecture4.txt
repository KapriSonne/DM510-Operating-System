cmd commands:
-----------------

$man             // shows manual
$man             // read(2) is syscalls
$man man         // gives all the different types of calls
$strace          // show all syscalls on a program

$setfacl -m u:username:r filename   // this instead of chmod<xxx>
$getfacl filename                   //
$top                                // see processes running
$finger <username>                  //
$df                                 // filesystem
$df -H                              // -H makes is human readable
$lsmod                              // list all modules loaded
